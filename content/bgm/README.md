# BGM Tracks for TikTok Videos

TikTok videos need background music. Place 15-20 second loop-friendly .mp3 tracks here.

## Required Tracks

| Filename | Style | Duration | Use Case |
|----------|-------|----------|----------|
| `bgm_calm.mp3` | Calm lo-fi | 15-20s loop | あるある系・感情的な投稿。夜勤明けや日常の静かな共感シーン |
| `bgm_upbeat.mp3` | Upbeat motivational | 15-20s | 転職成功・ポジティブ系。給与アップや新しい職場への期待感 |
| `bgm_emotional.mp3` | Emotional piano | 15-20s | 感動系・深い共感。患者とのエピソードや看護師の矜持 |
| `bgm_chill.mp3` | Chill acoustic | 15-20s loop | 地域ネタ・ライフスタイル系。神奈川の風景や通勤シーン |
| `bgm_energetic.mp3` | Energetic pop | 15-20s | トレンド便乗・明るい系。バズ狙いやチャレンジ系投稿 |

## Current Status

Placeholder sine-wave tones are generated by `scripts/download_bgm.py` so the pipeline does not break.
Replace these with royalty-free tracks when available.

## Royalty-Free Sources

- [Pixabay Music](https://pixabay.com/music/) (Free, no attribution required)
- [FreePD](https://freepd.com/) (Public domain)
- [Uppbeat](https://uppbeat.io/) (Free tier, attribution required)
- [YouTube Audio Library](https://studio.youtube.com/channel/UC/music) (Free with YouTube account)

## Notes

- Keep tracks under 20 seconds for TikTok slideshow alignment (total video ~15.5s)
- Loop-friendly: ensure the end transitions smoothly back to the beginning
- Avoid tracks with vocals (they compete with on-screen text)
- ffmpeg mix command: `ffmpeg -i video.mp4 -i bgm_calm.mp3 -filter_complex "[1:a]volume=0.3[bg];[0:a][bg]amix" -shortest out.mp4`
