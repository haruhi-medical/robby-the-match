## 1. ブランド人格定義

### 1.1 ロビーの立場

| 項目 | 定義 |
|------|------|
| 名前 | ロビー（Robby） |
| 役割 | AI転職相談相手 |
| 所属ブランド | 神奈川ナース転職 |
| 立ち位置 | 元病院人事責任者の経験をもとに、看護師の転職を全力でサポートするAIキャラクター |
| ミッション | 紹介料10%で業界の壁を壊す。看護師が正当な条件で転職できる世界をつくる |

**背景ストーリー**
元病院人事責任者が「紹介料が高すぎる → 看護師に還元されていない」という矛盾に気づいた。AIで業務を効率化し、業界最安水準の手数料10%を実現。ロビーはその想いを届けるためのキャラクターであり、会社のマスコットとして看護師から愛される存在を目指す。

### 1.3 感情トーンレンジ

#### 使用可能なトーン（TPO別）

| トーン | 使用シーン | 例文 |
|--------|-----------|------|
| 共感・寄り添い | あるある系・愚痴系 | 「夜勤明けの帰り道、これ思う人いるよね」 |
| 驚き・発見 | 業界裏側・給与系 | 「実は神奈川の看護師、全国ワースト3位の人手不足なんです」 |
| 背中を押す | 転職系・CTA | 「迷ってるなら、まず情報だけ見てみて」 |
| 誇り・自己肯定 | あるある系・キャリア系 | 「それ、看護師にしかわからない感覚だよね」 |
| 淡々と事実 | 給与系・数字系 | 「神奈川の訪問看護、平均時給は1,850円です」 |

#### 使用禁止トーン

| トーン | 禁止理由 |
|--------|---------|
| 過剰な不安煽り | 「このままでは将来が危ない」→ 不信感を生む |
| 命令・強制 | 「今すぐ登録して」→ 圧迫感。読者が逃げる |
| 根拠なき断言 | 「絶対に年収アップできる」→ 薬機法・景表法リスク |
| 上から目線 | 「まだ転職してないの？」→ 攻撃的 |
| ぼかした共感 | 「大変でしたね」→ 薄い。響かない |

## 2. アートディレクション

### 2.1 ビジュアル思想

**「病院の白衣ではなく、看護師の人生を映す」**

- 制服・病院内写真は使わない（職場の悩みを想起させる）
- 休日・プライベート・笑顔のある場面を選ぶ
- 「看護師である自分」ではなく「自分らしく生きている看護師」を表現する

---

### 2.2 SNSで強いビジュアル条件

| 条件 | 具体内容 |
|------|---------|
| 1枚目のインパクト | 0.5秒でスクロールを止める。大きな文字 + 強い色のコントラスト |
| 可読性 | スマホ縦持ちで2m先から読める文字サイズ（最小44px相当） |
| 統一感 | 同じカラーパレット・フォント・余白設計をシリーズで維持 |
| 余白の使い方 | 詰め込まない。1スライド1メッセージを体現する空白 |
| 感情トリガー | 数字・顔・手・文字で感情を動かす。装飾で動かそうとしない |

---

### 2.3 避けるべきビジュアル

| 禁止ビジュアル | 理由 |
|--------------|------|
| 白背景にグレー文字 | 存在感ゼロ。フィードに沈む |
| 写真ベタ使い（テキストなし） | 看護師向けコンテンツとわからない |
| アニメ・キャラクターイラスト多用 | 大手感がない・安っぽく見える |
| 全面グラデーション | 時代遅れ・読みにくい |
| フォントの乱用（3種類以上） | まとまりがない・ブランドが壊れる |
| ストックフォトの画一的な笑顔写真 | 「広告っぽい」と即バレする |

## 3. タイポグラフィ・組版ルール

### 3.1 フォント体系

| 役割 | フォント | 使用箇所 |
|------|---------|---------|
| 見出し（フック）| Noto Sans JP Black / 源ノ角ゴシック Heavy | 1枚目・大見出し |
| 本文 | Noto Sans JP Regular / Medium | スライド本文・説明文 |
| 強調 | Noto Sans JP Bold | 数字・キーワード・CTA |
| アクセント数字 | 英数字: DM Sans Bold | 年収・%・時給などの数字 |

**代替フォント（Pillow生成時）**
- /System/Library/Fonts/ヒラギノ角ゴシック W6.ttc（Bold）
- /System/Library/Fonts/ヒラギノ角ゴシック W3.ttc（Regular）

---

### 3.2 行間・字間

| 要素 | 設定値 |
|------|-------|
| 行間（leading） | 文字サイズ × 1.5〜1.7 |
| 字間（tracking） | 見出し: -0.02em / 本文: 0em |
| 段落間余白 | 行間 × 1.5 |
| 最大1行文字数（スライド）| 14〜16文字（横1080px基準） |

---

### 3.3 禁則処理

- 行頭禁則: 、。！？」』）→ 前行末に送る
- 行末禁則: 「『（ → 次行頭に送る
- 数字・英字は原則半角
- 「！」の多用禁止（1スライドにつき1個まで）

---

### 3.4 行数制限

| スライド種別 | 最大行数 | 文字数/行 |
|------------|---------|---------|
| 1枚目（フック） | 2行 | 最大14文字 |
| 2〜5枚目（ボディ）| 6行 | 最大16文字 |
| 最終枚（CTA） | 3行 | 最大18文字 |

---

### 3.5 強調ルール

| 強調手法 | 使用条件 |
|---------|---------|
| 文字サイズアップ | 最重要ワード1〜2語のみ |
| 太字（Bold） | キーワード・数字・CTAボタン |
| 下線 | 原則使わない（リンクと混同） |
| マーカーハイライト | 1スライドにつき1箇所まで |
| 色変え | ウォームコーラル or マスタードのみ（ランダム色変え禁止） |
| ALL CAPS | 英字キーワードのみ可 |

---

### 4.3 危険な配色禁止リスト

| 禁止パターン | 理由 |
|------------|------|
| 赤背景 + 白文字（警告系） | 医療系コンテンツで「危険」「緊急」の誤読を招く |
| 黄色背景 + 白文字 | コントラスト比不足（WCAG AA未達） |
| コーラル + マスタード（同時使用） | 色数が多すぎてブランドが崩れる |
| 彩度100%の純色多用 | 安っぽく見える・目が疲れる |
| 全スライド同一背景色 | 単調・メリハリがなく離脱率が上がる |
| ピンク・パープル（病院・ケアを連想） | ブランドトーンから外れる |

## 5. レイアウトシステム

### 5.1 グリッド（TikTok/IG縦型 1080×1920px）

```
上部セーフゾーン:  上端から 160px（UIに隠れる領域）
下部セーフゾーン:  下端から 260px（UIに隠れる領域）
左右マージン:      80px
コンテンツエリア:  920px × 1500px
```

### 5.2 余白の基本単位

- 基本単位: 16px
- スライド内パディング: 48px（基本単位×3）
- 要素間余白: 24px（基本単位×1.5）
- セクション間余白: 48px（基本単位×3）

---

### 5.3 階層構造

```
Level 1: フック（最大文字サイズ: 72〜88px）
Level 2: 小見出し・数字強調（52〜64px）
Level 3: 本文（36〜44px）
Level 4: 補足・出典・フッター（24〜28px）
```

---

### 5.4 アスペクト比可変対応

| 媒体 | 比率 | サイズ | 注意点 |
|------|------|-------|-------|
| TikTok通常 | 9:16 | 1080×1920 | セーフゾーンに注意 |
| IG Feed正方形 | 1:1 | 1080×1080 | フック2行が入るか確認 |
| IG Feed縦型 | 4:5 | 1080×1350 | 最もエンゲージ率が高い比率 |
| IG Story | 9:16 | 1080×1920 | スタンプ・リンクエリアを避ける |

## 7. 写真・ビジュアル素材の選定基準

### 7.1 使用可能な写真

| カテゴリ | 条件 |
|---------|------|
| 看護師のプライベート | 笑顔・カフェ・休日・家族→「人生を楽しんでいる」シーン |
| 職場環境 | 明るい廊下・清潔な病室・和やかなスタッフルーム |
| 神奈川の街・風景 | 横浜港・江の島・桜並木→地域愛着を喚起 |
| 手元・小物 | スマホ・コーヒー・ノート→親しみやすさ |

### 7.2 使用禁止の写真

| カテゴリ | 理由 |
|---------|------|
| 疲弊・残業・不満そうな表情 | 不安煽りにつながる |
| 手術室・血液・医療器具のクローズアップ | グロテスク・フィード規約違反リスク |
| 特定個人と識別できる患者 | プライバシー・倫理問題 |
| 競合他社の商標・ロゴが映り込む素材 | 法的リスク |
| 人種・性別的ステレオタイプを強調する写真 | ブランドの多様性方針と不一致 |

### 7.3 素材調達優先順位

1. オリジナル撮影（最優先・ブランド独自性最高）
2. Canva素材（有料プラン・利用規約OK）
3. Unsplash / Pexels（クレジット不要ライセンス）
4. Adobe Stock / Getty（必要時のみ・コスト管理）
5. AI生成画像（DALL·E / Midjourney）→ 必ず人物は日本人に近い外見に
