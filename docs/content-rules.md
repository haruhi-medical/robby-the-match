# 神奈川ナース転職 — コンテンツ生成ルール詳細版

> バージョン: 1.0 / 2026-03-16
> 適用範囲: TikTok・Instagram・LINEの全コンテンツ生成

---

## 1. コピーライティング基準

### 1.1 文体設計

| 媒体 | 基本文体 | 人称 | 文末 |
|------|---------|------|------|
| TikTok | 話し言葉・体言止め | ロビー → 看護師読者 | 「〜だよね」「〜です」「〜かも」 |
| IG Feed | 丁寧語ベース | 同上 | 「〜です」「〜ます」「〜でした」 |
| IG Story | 短文・タメ口寄り | 同上 | 「〜！」「〜？」「〜かも」 |
| LINE | 1対1対話 | ロビー → あなた | 「〜ですよ」「〜どうですか？」 |

**基本原則**
- 読む人は「ミサキ28歳・病棟勤務・転職を漠然と考えている」
- 「教える」ではなく「一緒に考える」スタンス
- 長い文より短い文。短い文より1単語の余白

---

### 1.2 強さと優しさのバランス

```
強すぎる（NG）  ←——————→  優しすぎる（NG）
「今すぐ決断せよ」           「大変でしたね」
「知らないと損する」          「あなたのことを応援しています」
「このままではまずい」         「いつでもお気軽に」

              ↓ 正しいゾーン

「神奈川の看護師、実は転職タイミングを知らずに損してる人が多い」
「迷ってる間も、あなたの市場価値は変わってるよ」
「まず情報だけ見てから判断してください」
```

---

### 1.3 断定口調ルール

| 状況 | ルール | 例 |
|------|--------|---|
| 数字・統計 | 根拠があれば断定OK | 「神奈川の看護師は77,188人、全国ワースト3位の人手不足」 |
| 年収・給与 | 「平均」「目安」「〜程度」を必ず付ける | 「横浜の病院看護師、平均年収は520万円程度」 |
| 転職結果 | 断定禁止。可能性・傾向で表現 | 「年収アップした人が多い」「改善ケースが増えている」 |
| 個人の状況 | 断定禁止。問いかけで表現 | 「もしかして、こんなこと感じてない？」 |
| 法規制・制度 | 必ず「〜の場合」「〜が一般的」で表現 | 「雇用保険は一般的に離職後1ヶ月程度で受給開始できます」 |

---

### 1.4 不安訴求の限界線

**許容範囲（事実ベースの問題提起）**
- 「神奈川の看護師、転職情報を持ってないだけで年収100万円差がつくことがある」
- 「夜勤手当の計算ミス、実は多い。確認したことある？」

**禁止（脅迫・過剰不安）**
- 「このまま働き続けたら体が壊れる」
- 「今の職場にいたら将来が危ない」
- 「転職しないと後悔する」
- 「知らない人は損してる」（→「知っている人が得している」に変換）

**判断基準**: 読んだ後に「怖い」ではなく「なるほど、考えてみよう」と思えるか

---

### 1.5 転職意欲別語り方

| 意欲レベル | 状態 | 語り方 |
|-----------|------|-------|
| 潜在層（意識なし） | 今の職場で普通に働いている | 「あるある」「業界裏側」で気づきを与える |
| 興味層（漠然と考え中）| たまに転職を考える | 「給与」「条件」の具体情報で現実を知らせる |
| 検討層（本気で考えている）| 求人を見たことがある | 「地域特化」「ステップ」で具体的行動を示す |
| 行動層（動こうとしている）| エージェントを探している | 「神奈川ナース転職の強み」「10%の理由」を伝える |

## 2. フック設計

### 2.1 フックの定義

**フック = スライド1枚目のメインテキスト**
- 役割: 0.5秒でスクロールを止め、次のスライドへの好奇心を生む
- 文字数: **25文字以内**（省略しすぎず日本語として自然な文に）
- 必須要素: **5W1H（誰が・何が・いつ・どこで・なぜ・どうした）のうち最低2要素**

---

### 2.2 フックの5構造パターン

#### パターン1: 数字から始まる（驚き型）
```
構文: [数字] + [対象] + [事実/問い]
例: 「神奈川の看護師77,188人の中で転職で年収アップした人の割合は？」
例: 「夜勤月8回の看護師、年収の差が最大120万円ある理由」
```

#### パターン2: 問いかけ型（共感型）
```
構文: [あるある状況] + [問い]
例: 「夜勤明けに転職サイト見てる神奈川の看護師、何人いる？」
例: 「ナースシューズ、職場のロッカーに2足ある人いない？」
```

#### パターン3: 逆説型（意外性）
```
構文: [一般認識] + 「でも実は」+ [意外な事実]
例: 「大病院=高給と思ってた。でも神奈川、中規模病院の方が高い場合がある」
例: 「夜勤なしの職場=楽、じゃないことを知ってる看護師は少ない」
```

#### パターン4: 未完了型（オープンループ）
```
構文: [状況説明] + [続きが気になる切り方]
例: 「横浜で転職した看護師が、年収より先に気にしたこととは...」
例: 「小田原の病院を選んだ理由、給与じゃなかった」
```

#### パターン5: 告白・体験型（親近感）
```
構文: [1人称的な告白/体験] + [共通の問い]
例: 「転職エージェントに連絡したら、翌日に同じ病院から電話が来た話」
例: 「履歴書を書こうとしたら3時間止まってた、あれ何だったんだろう」
```

---

### 2.3 フック品質基準

| 基準 | 合格条件 | 不合格例 |
|------|---------|---------|
| 5W1H | 最低2要素が明確 | 「看護師、知ってる？」（誰が何を？） |
| オープンループ | 疑問/未完了/理由/なぜ で終わる | 「看護師転職は神奈川ナース転職へ」（答えが出てる） |
| 文字数 | 25文字以内 | 「神奈川で働く20代の看護師が転職前に必ず確認すべき3つのこと」（長すぎ） |
| 事実ベース | 数字は根拠あり | 「年収500万円も夢じゃない！」（根拠なし） |
| 日本語として自然 | 省略しすぎない | 「看護師/転職/神奈川/年収/差」（キーワード羅列） |
| ブランドトーン | 不安煽り・命令形なし | 「このままでは取り残される看護師へ」（脅迫的） |

---

### 2.4 フックのNGとOK例

| NG例 | 問題点 | OK例 |
|------|--------|------|
| 「看護師さん必見！」 | 5W1Hゼロ・広告臭い | 「神奈川の夜勤ありの看護師、月収の計算合ってる？」 |
| 「転職で年収UP！」 | 断定・根拠なし・ループが閉じている | 「転職後に年収が上がった神奈川の看護師が選んだ条件とは」 |
| 「知らないと損する！」 | 脅迫型・広告臭い | 「転職エージェントが教えてくれないこと、ある」 |
| 「ロビーからのお知らせ」 | ブランド広告。スクロール止まらない | 「夜勤を月5回に減らして年収が上がった人の話」 |
| 「看護師転職するなら神奈川ナース転職！」 | 答えが出ている・購入CTA | 「神奈川の看護師が転職エージェントを選ぶ前に知っておきたいこと」 |

## 3. ボディ設計

### 3.1 1スライド1メッセージ原則

- 1枚のスライドに伝えることは**1つだけ**
- 「あれもこれも」は読者が混乱する → 離脱する
- 情報が2つあるなら、スライドを2枚に分ける

---

### 3.2 6枚構成4パターン

#### パターンA: あるある系（共感→安心→行動）
```
1枚目: フック（あるある状況を描写）
2枚目: 共感深化（「こんな気持ちになるよね」）
3枚目: 原因・背景（なぜそうなるのか）
4枚目: 反転・希望（「でも実は変えられる」）
5枚目: 具体策・情報（「こうすればいい」）
6枚目: CTA（「まず相談だけでも」）
```

#### パターンB: 給与・数字系（事実→比較→納得→行動）
```
1枚目: フック（数字で驚かせる）
2枚目: 現状の数字（根拠つき）
3枚目: 比較（エリア別・職種別・施設別）
4枚目: 差が生まれる理由
5枚目: 改善した人の例（匿名・傾向として）
6枚目: CTA
```

#### パターンC: 業界裏側系（秘密→暴露→信頼→行動）
```
1枚目: フック（「実は〜」「知られていない〜」）
2枚目: 業界の構造・仕組み
3枚目: 一般的な誤解
4枚目: 実態（数字・事実）
5枚目: 読者にとってのメリット
6枚目: CTA
```

#### パターンD: 転職ステップ系（問題→プロセス→安心→行動）
```
1枚目: フック（転職を考えている人への問いかけ）
2枚目: 転職でよくある不安
3枚目: ステップ1（まず何をするか）
4枚目: ステップ2（次に何をするか）
5枚目: 神奈川ナース転職の特徴（10%・地域密着）
6枚目: CTA
```

---

### 3.3 テキスト品質基準

| 項目 | 基準 |
|------|------|
| 1スライドの文字数 | 最大80文字（本文）。超えたら削る |
| 1スライドの行数 | 最大6行（ボディ）、2行（フック）|
| 読了時間 | 1スライド3秒以内で読み終わる量 |
| 難易度 | 中学卒業程度で理解できる語彙 |
| 専門用語 | 使う場合は必ず同一スライド内で説明 |
| 数字の使い方 | 半角。単位を必ず付ける（万円・人・%）|

## 4. CTA設計

### 4.1 ソフトCTA（全体の80%）

**定義**: 登録・問い合わせを直接求めず、次のアクションへの低いハードルを提示する

| 文言 | 使用シーン |
|------|----------|
| 「気になる人はプロフのリンクからどうぞ」 | あるある系・業界裏側系 |
| 「詳しい情報はプロフのリンクで見られます」 | 給与・数字系 |
| 「神奈川の求人、一度見てみて」 | 地域ネタ系 |
| 「ロビーに相談するだけでもOK」 | 転職不安系 |
| 「まず情報収集から始める人向けにまとめてます」 | 潜在層向け |
| 「コメントで聞いてくれれば答えるよ」 | エンゲージメント促進 |

---

### 4.2 ハードCTA（全体の20%）

**定義**: LINE追加・無料相談など、具体的なコンバージョンを求める

| 文言 | 使用シーン |
|------|----------|
| 「LINE無料相談、プロフのリンクから登録できます」 | 行動層向け・シリーズ最終回 |
| 「神奈川の求人を無料で見るにはプロフのリンクへ」 | 給与系投稿の最後 |
| 「手数料10%の理由、LINEで直接聞いてみて」 | 業界裏側系投稿の最後 |

---

### 4.3 CTAのNG例

| NG例 | 理由 |
|------|------|
| 「今すぐ登録！」 | 命令形・圧迫感 |
| 「無料だから損はない！」 | 安っぽい。価値を下げる |
| 「登録しないと求人を見れない！」 | 脅迫型 |
| 「3分で完了！」 | 根拠なし・安易 |
| 同一投稿にCTAが3回以上 | 過剰・うるさい |
| 「フォローとリポスト忘れずに！」 | 強要・SNSルール違反リスク |

## 6. カテゴリ別コンテンツ設計

### 6.1 MIX比率（現行）

| カテゴリ | 比率 | 月間本数（20本/月想定） |
|---------|------|---------------------|
| あるある | 35% | 7本 |
| 給与・数字 | 20% | 4本 |
| 業界裏側 | 15% | 3本 |
| 地域ネタ | 15% | 3本 |
| 転職ステップ | 10% | 2本 |
| トレンド | 5% | 1本 |

---

### 6.2 あるある（35%）

**目的**: 潜在層・興味層への認知獲得・共感獲得

**設計方針**
- 「これ私のこと！」と思わせる具体的な描写
- 職場・日常・キャリアの3軸でネタを作る
- 解決策は不要。「あなたのことを理解している」メッセージが重要

**ネタ例（職場軸）**
- 夜勤明けの帰り道の感覚
- 申し送りノートの書き方が先輩によって全然違う問題
- 急変対応後にまた普通の業務に戻る感覚
- インシデントレポートを書く時の気持ち

**ネタ例（日常軸）**
- 休日に仕事のことが頭から離れない
- 友達への職業説明の難しさ
- 健康診断の結果に敏感になりすぎる
- 飲み会で「看護師なんだ、○○が〜で」と言われる感覚

**ネタ例（キャリア軸）**
- 5年目なのにまだ転職を考えている自分
- 「スキルを磨く」と言いながら何をしたらいいかわからない
- 認定看護師を目指すかどうか迷っている

---

### 6.3 給与・数字（20%）

**目的**: 興味層への市場理解促進・転職動機形成

**設計方針**
- 数字は必ず根拠を示す（ハローワークデータ・厚労省・自社調査等）
- 比較で驚きを生む（エリア別・施設別・年齢別）
- 「高い/低い」ではなく「なぜその差が生まれるか」まで説明する

**数字の根拠データ（参照可能）**
- 神奈川県看護師: 77,188人（全国ワースト3位の人手不足）
- 応援ナース時給: 3,000円（目安）
- ハローワーク神奈川看護師求人: 1,364件
- 訪問看護師平均時給: 1,850円（目安）

**コンテンツ例**
- 「横浜vs川崎、看護師の平均年収比較」
- 「夜勤手当の計算方法を間違えていた話」
- 「神奈川の訪問看護、時給の実態」
- 「年収500万円の看護師が選んだ施設の種類」

---

### 6.4 業界裏側（15%）

**目的**: 信頼獲得・専門性の証明・神奈川ナース転職の差別化

**設計方針**
- 大手エージェントが言わないことを言う
- 「手数料10%の理由」を自然に折り込む
- 「業界の構造を知っている = 信頼できる」印象を作る

**コンテンツ例**
- 「転職エージェントの手数料、看護師には関係ない？実は関係ある」
- 「求人票に書いてある残業時間、信じていい？」
- 「転職後に後悔した人がやっていたこと」
- 「面接で聞いてはいけないことと、聞いていい言い方」

---

### 6.5 地域ネタ（15%）

**目的**: 神奈川県内での検索流入・地域密着感・大手との差別化

**設計方針**
- 大手が絶対にやらない地域特化を強みにする
- 地名を必ずタイトル・スライドに入れる（#横浜 #川崎 #平塚 等）
- 「神奈川の看護師にしか刺さらない」コンテンツを作る

**コンテンツ例**
- 「横浜で働く看護師、18区でこんなに違う通勤事情」
- 「小田原の病院、実は条件がいい理由」
- 「湘南エリアの訪問看護、需要が急増している」
- 「川崎の夜勤手当、横浜と比べてみた」
- 「相模大野から横浜の病院に通っている看護師の話」

**地域カバレッジ**

## 8. プラットフォーム別最適化

### 8.5 プラットフォーム横断の原則

- **同一コンテンツの流用可**。ただしキャプション・ハッシュタグは媒体に合わせて書き直す
- **TikTok → IG Reel**: 解像度は同じ。TikTokウォーターマークを消してから投稿
- **IG Feed → IG Story**: 縦型に合わせてリサイズ。テキストは短縮する
- 各媒体の成果（エンゲージ率・保存率・フォロワー増加）を週次で比較し、成果が出た媒体に優先配分する
