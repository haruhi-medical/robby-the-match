## コンテンツクラスター設計

```
ピラー: /lp/job-seeker/（神奈川 看護師 転職）
├── エリア軸: area/yokohama.html → 区単位ページ群
├── 条件軸: guide/（夜勤なし/パート/復職/年収）
└── 年収軸: guide/salary-kanagawa.html → 地域別年収ページ群
```

## 内部リンク設計

```
メインLP → 人気エリアTop5 + ガイドTop3
各エリア → 隣接エリア5件 + 関連ガイド3件
各ガイド → 関連ガイド4件 + 該当エリア4件
各ブログ → 関連ガイド2件 + 関連エリア2件
```
