## 登録フロー（Google側）

### Step 1: アカウント確認
- アクセス: <https://business.google.com/create>
- ログイン: `robby.the.robot.2026@gmail.com`（.env参照）

### Step 2: ビジネス情報入力

| フィールド | 入力値 | 備考 |
|-----------|-------|------|
| ビジネス名 | `神奈川ナース転職` | |
| カテゴリ（主） | `人材紹介業` | 英: Employment Agency |
| カテゴリ（追加） | `求人情報提供サービス` / `看護師` | 最大9個登録可、SEO強化に有効 |

### Step 3: 所在地の質問 →「いいえ、店舗や事務所はありません」を選択

**⚠️ 重要**: 「はい、住所を追加します」を選ばない。
→ サービスエリアビジネスとして登録される。

### Step 4: サービス提供エリア

複数地域を入力可能。以下を登録:

- 神奈川県（全域）
- 横浜市 / 川崎市 / 相模原市 / 横須賀市 / 湘南 / 県西 / 県央
- （将来拡大時）東京都 / 千葉県 / 埼玉県

### Step 5: 連絡先

| フィールド | 入力値 |
|-----------|-------|
| ウェブサイト | `https://quads-nurse.com/` |
| 電話番号 | 社長判断（LINE中心のため未入力でも可） |

**電話番号登録の判断ポイント**:
- 入力しない: LINE完結方針と整合、誤電話リスクなし
- 入力する: GBP表示でクリック通話CV計測可能、ただし電話対応体制必要

→ **推奨**: 未入力（LINE完結で通し、将来 IVR 導入時に追加）

### Step 6: 認証方式選択

サービスエリアビジネスは通常 **ハガキ認証** のみ:

- ハガキ送付先: 事業所住所（GBP上は非公開のまま内部で保持）
- 到着: 約2週間
- ハガキ記載の5桁コードをGBPに入力して完了

## 登録後の設定（認証完了を待たずにできる）

### 営業時間

```
月-金: 09:00 - 21:00
土日祝: 10:00 - 18:00
```

※ LINE応答時間に合わせて設定。24時間表記はせず、現実の返信帯で正直に。

### ビジネス説明（750字以内）

```
神奈川県を中心とした看護師専門の転職サービスです。LINEで完結する
転職サポートを提供しており、電話での営業連絡は一切行いません。
転職に関する相談、求人紹介、職務経歴書作成のサポートまでAIと担当者が
連携して対応いたします。

有料職業紹介事業許可番号: 23-ユ-302928
```

**⚠️ 注意**:
- 手数料率「10%」は**書かない**（MEMORY.md `feedback_gbp_address.md` 関連判断）
- 代表者名・社名は**書かない**
