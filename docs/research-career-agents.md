## 0. エグゼクティブサマリー（社長が10秒で理解できる）

**キャリアエージェントとは何か**: 業界は11ステップに標準化されている。1日80〜100件の架電ノルマを背負ったCAが、月150〜200万円の粗利ノルマを追いかけ、求職者を「希望無視で推薦」「しつこい電話」「おとり求人」の地獄に叩き込む労働集約モデル。

**業界の矛盾**: 大手CAの年収天井は550万円で過酷。求職者の8割が「しつこい」と不満。76.5%が「おとり求人」被害。それでも病院は年収の30-35%（看護師1名で100-150万円）の手数料を払っている。

**我々が勝てる理由**:
1. **11ステップのうち8ステップはAIで代行可能**（ヒアリング・マッチング・書類・面接対策・日程調整・FAQ・リマインド・要約）
2. **人間が必要な3ステップ**（施設の内部情報・交渉・感情的サポート）は、神奈川特化の「浅く広く」ではなく「狭く深く」で差別化可能
3. **日本に看護師特化AI代行は存在しない**（ROXX・doda・リクルートは全て横断型）
4. **既存資産**: LINE Bot（intake_light 3問 + Q-flow 10問 + AI相談 + 匿名打診）+ 212施設DB + ハローワーク1,364件 + Cloudflare Worker。土台の8割はすでにある。

**結論**: 我々が作るべきは「看護師AIキャリアエージェント」。Phase 1（現LINE BotのP0修正 + 書類生成AI）で3ヶ月以内に MVP。Phase 2（音声AI + 給与交渉叩き台）で業界破壊開始。

## 1. キャリアエージェントの業務全容（11ステップ）

5社（レバウェル / マイナビ / 看護roo! / ナースではたらこ / 医療ワーカー）を比較した結果、業界は以下11ステップにほぼ完全に標準化されている。

| # | ステップ | 5社共通の実施内容 | 所要時間/回数 |
|---|---------|------------------|-------------|
| 1 | **登録** | Web or 電話、1分以内、完全無料 | 1回・1分 |
| 2 | **初回面談（電話/Web/対面）** | 登録後1〜2営業日内にCAが架電、30〜60分 | 1回・30-60分 |
| 3 | **ヒアリング** | 希望条件+転職理由+キャリアプラン+不安+連絡可能時間 | 1回・面談内 |
| 4 | **求人紹介** | 公開+非公開求人を複数提示、LINE/メール/電話 | 複数回・随時 |
| 5 | **応募書類作成** | 履歴書・職務経歴書の添削、志望動機作成支援 | 1-3回 |
| 6 | **応募・面接調整** | 応募手続きと面接日程調整をCAが代行 | 複数回 |
| 7 | **面接対策** | 応募先ごとの想定質問、模擬面接、マナー指導 | 1-2回 |
| 8 | **面接同行 or リモート伴走** | 4社に同行オプション（マイナビのみ明示なし） | 希望時 |
| 9 | **条件交渉** | 給与・勤務条件・入職日の交渉代行（全社共通） | 1-2回 |
| 10 | **退職交渉サポート** | 退職意向の伝え方・引継ぎ・スケジュール指導 | 随時 |
| 11 | **入職・定着フォロー** | 入職後の悩み相談（看護roo!のみ当日/数日後/1週/1ヵ月/3ヵ月と明示）| 3-5ヶ月 |

### 共通ヒアリング項目（11項目で9割カバー）

保有資格 / 勤務地エリア / 勤務形態（常勤・非常勤・夜勤有無） / 希望年収 / 希望休日数 / 転職理由 / 転職希望時期 / キャリアプラン / ライフプラン（子育て・結婚） / 現職の悩み / 連絡可能な時間帯

### 差別化軸の比較

| 項目 | レバウェル | マイナビ | 看護roo! | ナースではたらこ | 医療ワーカー |
|------|-----------|---------|----------|----------------|-------------|
| 独自強み | **年4,000回の職場訪問** | 8ステップ明示+お客様相談室 | **入職後5回の定期連絡**（当日/数日後/1週/1ヵ月/3ヵ月）| **逆指名制度 + 24h受付** | **地方での面接同行 + 採用条件確認書** |
| 面接同行 | オプション | 不明 | 有（地方弱い） | 一部地域のみ | 有（地方強い） |
| 手数料 | 30-35% | 30-35% | 30-35% | 30-35% | 30-35% |

**→ 「現場の足で稼ぐ内部情報」と「同行」だけが真の人的価値。他は全てAI化可能。**

## 2. CAの業務実態 — なぜ業界は求職者に嫌われるか

### CAの1日（大手標準）

| 時刻 | 業務 | 量 |
|------|------|------|
| 8:00-9:30 | 出社・朝会・メール/LINE対応 | 前日夜間メッセージ返信 |
| 9:30-12:00 | 新規架電ラッシュ | **40-50件/午前** |
| 13:00-16:00 | 書類作成・RA連携・求人ピックアップ | 3-5件推薦/求職者 |
| 16:00-18:00 | 求人提案電話 | 継続架電 |
| 18:00-21:30 | **就業後面談ラッシュ**（看護師は18時以降に集中）| **5-6件/日** |
| 21:30- | CRM入力・推薦文作成・翌日準備 | 〜22時 |

### KPI・ノルマ（新人〜中堅）

| 項目 | 水準 |
|------|------|
| 架電ノルマ | **80-100件/日** |
| アポ獲得率 | 1-3.5%（新人） |
| 月間粗利ノルマ（新人） | 100-150万円 / **月1-2名成約** |
| 月間粗利ノルマ（若手） | 150-200万円 / **月2-3名成約** |
| 同時進行アクティブ求職者 | 100-200名 |
| 月間新規面談 | 30名 |
| 登録→成約率 | 5-10% |

### 過酷労働の構造

- レバウェル営業職: 残業 **51.4時間/月**、固定残業 **80時間込み**、年収天井 **550万円**
- 看護師1人成約で手数料 **100-150万円**（年収の30-35%）→ この高額手数料がノルマ圧の源泉
- 人材紹介経由の准看護師 **1年以内離職率41.5%**（GemMed）。強引な推薦で入ってすぐ辞める構造
- 厚労省調査: 1,100事業所の **60%が法令違反**。都内病院の紹介会社満足度「不満」**7割**
- **2025年4月施行**: 祝い金全面禁止 + 手数料公開義務化 → 業界激震中

### 求職者の被害（定量）

| 不満 | 発生率 | 典型 |
|------|--------|------|
| 電話がしつこい | ほぼ100%認識 / 8割が実体験 | 1日5回以上、SMS/メール/LINE/電話4チャネル攻撃、番号変えて着信拒否突破 |
| おとり求人 | **76.5%が違う求人案内、74.4%が「募集終了」** | 電話を取らせるための囮 |
| 転職後ギャップ | **約80%** | 「残業少ない」→多い、「アットホーム」→人間関係悪 |
| 希望無視ゴリ押し | 頻出 | 「2日で履歴書提出」「考えが甘い」 |
| 辞退時引き止め | 頻出 | ノルマ達成のための粘り |

**根本原因**: 1件80-150万円の手数料 × 月2名ノルマ = 「架電100件、希望無視で推薦」の労働集約構造。

| 10 | 退職交渉サポート | **70% AI** | 退職届テンプレ、引き止め対応スクリプト、労基/弁護士提携先紹介 | Phase 2 |

- 8ステップ（1,2,3,4,5,7,11, 6の一部）= **完全AI化可能** → 人件費を90%カット

- 1ステップ（8）= **人間必須** → 希望者のみ、神奈川内なので低コスト

これが「手数料10%」を成立させるコスト構造の根拠。

## 4. 既存資産とのギャップ分析

### 既に実装済み（LINE Bot worker.js）

| 機能 | 現状 | カバー範囲 |
|------|------|----------|
| 登録 | LINE友だち追加 → welcome 4択 | ステップ1 ✅ |
| intake_light（3問）| エリア・働き方・温度感 | ステップ3の30% |
| Q-flow（10問） | urgency別にFULL/MEDIUM/SHORT分岐 | ステップ3の90% |
| 匿名打診フロー | apply_info → career_sheet → 匿名プロフィール生成 | ステップ4+6の一部 |
| マッチング | matching_preview で3件 → matching_browse で追加 | ステップ4 ✅ |
| AI相談 | OpenAI GPT-4o-mini（Workers AIフォールバック）5-8ターン | ステップ2,3の一部 |
| 逆指名 | 病院名指定 → 打診 | 独自機能 |
| ナーチャリング | Day3/7/14/30 自動Push | ステップ11の一部 |
| Slack引き継ぎ | handoff → Slack転送 → !reply で人間応答 | AI→人間の切替 ✅ |
| 施設DB | config.js 212施設（全神奈川） | 基盤 ✅ |
| 求人DB | ハローワークAPI 1,364件、16エリア123件表示 | 基盤 ✅ |

### 足りないもの（AI代行システムとして完成させるには）

#### 🔴 P0: 既知の致命的バグ（LINE_BOT_FLOW_AND_ISSUES.md より）

1. **intake_light で匿名プロフィールがほぼ空白** — 3問しか聞かないため「資格・経験年数・スキル・転職理由」が全部「（未回答）」。病院に打診するプロフィールとして機能しない
2. **AI相談のターン延長が無限ループ** — 8ターン到達後 `consult=extend` タップしても上限8のまま
3. **「応募完了！書類選考3営業日」の嘘メッセージ** — 匿名打診フローなのに事実と異なる文言が残存
4. **area_page経由のwelcome 4ボタンが全て同じpostback** — workStyleが記録されない
5. **OpenAI APIキー失効時に完全無応答** — Push APIが失敗するとユーザーに何も届かない

→ **AIエージェントとして機能させるには、まずこの5つを全部潰す必要がある**

#### 🟡 P1: 新規実装が必要な機能

1. **AI書類生成**（Phase 1 最優先）
   - 履歴書・職務経歴書を最短1分で生成（doda実証済み）
   - Q-flow の回答 + 施設DB → GPT-4o でPDF生成
   - 書類通過率+3pt、面接進出+30% の実績データあり

2. **AI面接対策**（Phase 2）
   - 応募施設の特徴 + よくある質問 → 想定Q&A生成
   - 模擬面接（ROXX方式の音声 or テキスト）

3. **AI面接日程調整**（Phase 2）
   - Google Calendar連携で候補日提示 → LINE確定
   - Paradox方式、80%が5分以内完了

4. **入職後フォロー自動化**（Phase 1）
   - 看護roo! 方式: 入職当日/数日/1週/1ヵ月/3ヵ月の自動Push
   - ミスマッチ検知 → 人間介入トリガー

5. **「おとり求人ゼロ」保証**（Phase 1）
   - ハローワーク求人の実在検証（APIで定期更新済み → 削除求人を即座に反映）
   - 施設DB の最終更新日を表示

6. **音声AI対応**（Phase 3）
   - OpenAI Realtime API（ROXXが採用）
   - 夜勤明けで「文字を打つのもだるい」看護師向け

7. **交渉叩き台生成**（Phase 2）
   - 給与・夜勤回数・休日数の交渉文面をAIが作成 → 人間が送信
   - TopResume「AI 90% + 人間10%」の黄金比

#### 🟢 P2: 将来の拡張

- 施設口コミDB（神奈川212施設の匿名口コミを蓄積）
- 退職届テンプレート + 労基/弁護士提携先紹介
- 医療広告ガイドライン準拠の自動チェック
- ペアリング学習（成約結果のフィードバックでマッチング精度向上）

## 7. AI代行システム 実装ロードマップ

### Phase 1: 現LINE Botの完成 + 書類AI（〜2026-06）

**目標**: P0課題を全て潰し、匿名プロフィールが機能する状態にする。書類生成AIを追加。

1. **P0-1 修正**: intake_light を拡張し、匿名プロフィールに必要な7項目（資格・経験年数・診療科・転職理由・希望年収・希望休日・現職悩み）を追加収集
2. **P0-2,3,4,5 修正**: ターンループ解消、嘘メッセージ削除、postback修正、OpenAI失敗時のフォールバック強化
3. **AI書類生成**: Q-flow回答 → GPT-4o で履歴書・職務経歴書・志望動機をPDF化 → LINEで送信
4. **入職後フォロー自動化**: Cloudflare Worker Cron で5回Push（当日/数日/1週/1ヵ月/3ヵ月）
5. **おとり求人ゼロ保証**: ハローワーク求人の更新日・削除検知をUIに表示

**推定コスト**: OpenAI API ¥5,000-15,000/月（100人分のヒアリング+書類生成想定）

### Phase 2: AI面接対策 + 日程調整 + 交渉叩き台（〜2026-09）

**目標**: 応募〜内定までの「面接周辺」を全AI化

1. **AI面接対策**: 施設特徴 + 過去問DB → 想定Q&A生成、模擬面接（テキスト）
2. **AI日程調整**: Google Calendar API + LINE で候補日提示、Paradox方式
3. **交渉叩き台生成**: 給与・夜勤・休日の交渉文面をAIが作成 → 人間承認 → 送信
4. **退職交渉支援**: テンプレ + スクリプト + 提携先（労基/弁護士）紹介

**推定コスト**: +¥10,000-20,000/月

### Phase 3: 音声AI + 学習ループ（〜2026-12）

**目標**: 夜勤明け・隙間時間の音声対応 + 成約データ学習

1. **OpenAI Realtime API**: 音声でヒアリング・相談（ROXX方式）
2. **学習ループ**: 成約/離脱データをフィードバック、マッチング精度向上（autoresearchの手法を流用）
3. **施設口コミDB**: 入職後フォローから内部情報を逆流収集（看護roo!の仕組みを模倣）

**推定コスト**: +¥30,000-50,000/月（Realtime APIは高額）

## 12. 結論：なぜ我々が勝てるのか（3行）

1. **大手CAは1日100件架電のノルマで疲弊し、求職者に嫌われている**。この構造は手数料30%を支えるため不変
2. **日本に看護師特化AIキャリアエージェントは存在しない**。既存AIは全て横断型で業界知識が浅い
3. **我々は既にLINE Bot + 212施設DB + ハローワーク連携 + Slack引き継ぎを持っている**。Phase 1の拡張で MVP に到達し、手数料10%で業界破壊が始まる

**ミサキ（28歳看護師）の動きが変わる**: 「しつこい電話に消耗」→「LINEで匿名、いつでも相談、3件の厳選求人、書類はAIが1分、面接対策も万全、10%手数料で病院も喜ぶ」。
