# 画像生成→投稿 自動化パイプライン

## 推奨パイプライン（コスト最適版）

1. **トピック選定** — コンテンツMIX比率に従い選択
2. **台本生成（Claude API）** — 6枚構成テキスト、ペルソナチェック
3. **画像生成（gpt-image-1）** — テキストなし背景のみ、1024×1536縦型
4. **テキスト焼き込み（Pillow）** — 日本語オーバーレイ、フォント≧画像幅1/8
5. **キャプション生成** — 200文字以内、ハッシュタグ4個以内
6. **Slack通知→人間承認** — 画像+キャプション送信
7. **データ記録（24時間後）** — 再生数・いいね・保存・コメント

## 技術的注意点

- Cloudflare Workers AI画像 → Pillowで一度PNG再保存してからClaude APIに送る
- gpt-image-1の日本語テキストは不安定 → 必ず後乗せ（Pillow）
- ベース画像は使い回し可能（コスト削減）
- TikTok動画: H.264 1080x1920 30fps CRF18、15.5秒構成

## 月間コスト: ~1-2万円

- Claude API: ~5,000円
- gpt-image-1: ~5,000円（使い回しで半減可能）
- LINE公式: 無料〜5,000円

転職×AI: B01-B05（年収差/転職相談/適齢期/手数料/退職理由）

## A/Bテスト優先順位

1. フックのバリエーション（最もインパクト大）
2. 1枚目の文字数（20字 vs 10字）
3. 投稿時間（17:30 vs 21:00）
4. CTA配置（6枚目 vs 5枚目+6枚目おまけ）
