# 大手看護師人材紹介5社 業務フロー調査

> **作成日**: 2026-04-21
> **目的**: AIエージェント化に向けた、大手5社のキャリアアドバイザー業務フローの徹底抽出
> **調査方法**: 各社公式サイト・第三者レビュー記事のWebFetch/WebSearch
> **注意**: 「不明」と明記した項目は公式に記載が見つからなかった項目。推測は記載していない。

## 1. レバウェル看護（旧 看護のお仕事 / レバレジーズメディカルケア）

### サービスの流れ
公式発表の6ステップ（busiconet調査より）:

1. **登録**（所要1分・無料）
2. **ヒアリング**（電話で15分程度、希望条件と価値観を確認）
3. **求人紹介**（メールやLINEで複数求人提示）
4. **書類選考・面接**（書類添削と面接対策を実施）
5. **条件交渉・退職相談**（給与・休日などの交渉代行）
6. **就業開始・アフターフォロー**（入職後も継続サポート）

登録から数時間～数日で担当キャリアアドバイザーから電話連絡が入る。

### ヒアリング内容
公式「転職アドバイザーについて」ページおよび各種解説記事から抽出:
- 希望の勤務地・勤務時間等の条件
- 転職活動の不安
- 職務経歴
- 転職を考えた理由
- 給与・勤務地・働き方などの希望条件
- 将来のキャリアに対する考え
- 連絡可能な時間帯
- 現職の状況
- 子育てなどプライベート事情

### 求人紹介
- **保有求人数**: 業界最大級「29万件以上」（医療介護全体）/ 看護師単独では13万件超
- **非公開求人**: 12万件以上
- **紹介方法**: LINE・メールで複数求人を提示
- **紹介件数**: 明示なし（「条件に合う複数件」）
- **非公開求人の内訳**: 大学病院・総合病院、高収入、看護主任/看護師長、産業看護師など

### 書類サポート
- 履歴書・職務経歴書の作成アドバイス
- 「過去の面接をもとにした履歴書の添削」

### 面接対策
- 過去入職者から面接質問をリサーチして対策実施
- 志望動機のていねいな添削
- 模擬面接実施可能
- キャリアアドバイザーの**面接同行オプション**あり
- 面接当日は「リラックスできるようサポート」

### 企業対応
- 面接日程調整を代行
- 自分では言いにくい給与・労働条件の交渉を代行
- 条件交渉・退職相談まで一気通貫

### 内定後フロー
- 条件交渉・退職相談の代行
- 入職日調整

### 入職後フォロー
- 給料や人間関係の悩み、今後のキャリア相談に継続対応
- 期間の明示なし（「継続サポート」）

### 差別化ポイント
- **年間数千回～4,000回の職場訪問**: 医療方針・退職者の理由まで現場情報を収集（最大の差別化）
- LINEでのやり取りが可能
- 業界最大級の求人データベース

### 手数料（採用側・推定）
- 看護師人材紹介業界の相場: **年収の30～35%**（レバウェル固有の公表なし）

## 2. マイナビ看護師（マイナビ）

### サービスの流れ
公式「ご利用ガイド」の**8ステップ**:

1. **登録**（フォーム or 電話、迷い段階でも相談可）
2. **面談**（電話・メール・直接面談・Web面談のいずれか）
3. **求人紹介**（希望に合う求人をできるだけ数多く提示）
4. **応募先決定**（キャリアプランから一緒に選定）
5. **書類選考**（評価されやすい書き方のアドバイス）
6. **面接**（マナー・模擬面接・面接先に合わせた対策、日程調整代行）
7. **内定～入職**（入職条件確認、辞退連絡代行、退職支援）
8. **入職後**（お客様相談室スタッフがフォロー）

### ヒアリング内容
公式「初めての転職」ガイドおよびキャリアアドバイザー紹介ページより:
- 相談したい/情報収集したいと思ったきっかけ
- 仕事で不満に思っていること・悩んでいること
- 興味のある分野や好きな仕事
- 思い描く理想の職場
- 今何をやりたいのか
- 転職に対する不安
- ライフプラン・ワークライフバランス
- 長期的なキャリアプラン
- 転職希望時期・希望条件

### 求人紹介
- **非公開求人**: 多数保有（具体数の明示なし）
- **紹介方針**: 「できるだけ数多くご紹介」
- **情報内訳**: 勤務環境・福利厚生・職場の雰囲気も含む
- エリア専任制

### 書類サポート
- 履歴書・職務経歴書の書き方指導
- 経歴・自己アピールの表現指導
- 応募先に適した表現方法の提案

### 面接対策
- マナーなど基本事項
- **応募病院別の想定質問**を提供
- 面接官の人柄・人数の詳細情報を提示
- 希望時は**模擬面接**実施
- 服装選択など細部相談

### 企業対応
- 面接日程調整を代行
- 条件確認をアドバイザーが実施
- 「面接がうまくいかなくてもアドバイザーが面接先にフォロー」

### 内定後フロー
- 入職条件の確認を代行
- **辞退連絡を代行**
- 職場への伝え方・業務引継ぎ・退職スケジュール相談
- 退職意向の伝え方指導

### 入職後フォロー
- 「お客様相談室スタッフ」によるフォロー（CA本人ではない）
- 「入職直前、入職後の数か月」が対象

### 差別化ポイント
- 厚生労働大臣認可の職業紹介事業
- 医療・介護分野における適正認定事業者
- 「お客様相談室」による専門アフターフォロー
- 転職経験者による実践的アドバイス
- 8ステップの明確なフロー開示

### 手数料（採用側）
- 公式の公表なし。業界相場：年収の30～35%

## 3. 看護roo!転職（株式会社クイック）

### サービスの流れ
6ステップ:

1. **登録**（30～60秒で完了）
2. **初回連絡**（登録から2営業日以内、コールスタッフが登録内容確認）
3. **初回面談**（電話で実施、アドバイザーが詳細ヒアリング）
4. **求人紹介・書類添削・応募**
5. **面接対策・模擬面接・面接**
6. **内定・条件交渉・転職・入職後フォロー**

### ヒアリング内容
**2段階ヒアリング方式**:
1. コールスタッフによる登録内容確認
2. アドバイザーによる詳細ヒアリング

項目:
- キャリアの棚卸し
- 転職理由
- 希望給与
- 勤務時間
- 職場雰囲気の希望
- 転職で実現したいこと・譲れない条件
- 方向性・スケジュール

特徴: 「社内で繰り返しロールプレイングを行っているからヒアリング力が高い」

### 求人紹介
- **公開求人数**: 約24万件（2026年時点）
- **非公開求人**: あり（「好条件好待遇の案件」「入ってきたばかりの求人」「応募が集中する案件」を非公開化）
- **紹介手段**: 電話・メール・**LINE**
- エージェント求人と直接応募求人の2種類
- スカウト機能もあり（プロフィール登録で求人オファー）

### 書類サポート
- 履歴書・職務経歴書の添削（無料）
- 応募先が求める人物像を踏まえた指導
- より効果的な書き方・好印象を与える表現のアドバイス
- 志望動機の添削

### 面接対策
- **模擬面接**実施（質問回答方法・話し方・身だしなみ・マナー指導）
- 過去に面接した応募者から聞かれた質問をヒアリングし、的を射た対策
- 応募先ごとの面接アドバイス
- オンライン面接練習も提供
- 面接のフィードバック共有

### 企業対応
- 面接日程調整代行
- 応募先との交渉・やり取り代行

### 内定後フロー
- **面接同行あり**（拠点が少ないため地方では難しいケースあり）
- 面接時に聞きにくいことを代わりに聞く
- 年収交渉・入社日調整
- **内定辞退の代行**
- 円満退職アドバイス

### 入職後フォロー
- **明確な定期連絡スケジュール**:
  - 入職当日
  - 入職数日後
  - 入職1週間後
  - 入職1ヵ月後
  - 入職3ヵ月後
- LINEでやり取り可能
- トラブル時の対応

### 差別化ポイント
- **一気通貫システム**（求人紹介～入職後まで同じ担当）
- **医療機関の詳しい内部情報**（転職者アフターフォローで職場情報をリサーチ）
- **入職後の定期連絡**（当日～3ヵ月後まで5回）
- LINE対応
- 首都圏・東海・関西に特化

### 手数料（採用側）
- 公式公表なし。業界相場：年収の30～35%

**参考URL**:
- https://www.kango-roo.com/career/service/
- https://www.kango-roo.com/career/lp/hajimete/
- https://talentsquare.co.jp/career/kango-roo-reputation-nurse/
- https://digi-ato.com/content/article29/

## 4. ナースではたらこ（ディップ株式会社）

### サービスの流れ
公式「サービスの流れ」ページの**5ステップ**:

1. **STEP1 登録・相談**（スマホ・PC・電話、年中無休、約1分）
2. **STEP2 希望条件確認**（電話またはメール、現職に支障なく進められる）
3. **STEP3 求人紹介**（非公開求人含む。独自取材情報を提供）
4. **STEP4 面接サポート・条件交渉**
5. **STEP5 内定～入職後フォロー**（履歴書添削・面接対策・一部地域は同行）

### ヒアリング内容
登録フォーム項目:
- 転職目的
- 保有資格
- 希望勤務形態
- 求人探索時期
- 勤務状況
- 希望の働き方
- 郵便番号
- 氏名・生年月日
- 電話番号

電話面談でさらに詳しい希望条件を聞き取る。

### 求人紹介
- 日本最大級の看護師求人データベース
- **非公開求人あり**（Webには出していない好条件）
- 残業状況・休暇制度など**独自取材情報**を提供
- 聞きづらい確認事項はアドバイザーが代行確認

### 書類サポート
- 履歴書の書き方アドバイス・添削
- 職務経歴書の添削

### 面接対策
- 事前アドバイス・面接対策実施
- **一部地域では面接同行あり**

### 企業対応
- 面接日程調整代行
- 勤務条件・給与交渉代行
- 聞きづらい事項の代行確認

### 内定後フロー
- 入職前の相談対応
- 条件交渉

### 入職後フォロー
- 「あんしんサポート窓口」設置
- アドバイザー変更などの相談も可能
- 入職後の悩み相談まで対応

### 差別化ポイント
- **逆指名制度**（最大の特徴）:
  - 働きたい医療機関を指名できる
  - 求人が出ていない施設にもアドバイザーが条件確認しにいく
- **24時間365日受付**（年中無休）
- プライバシーマーク・ISO27001認証
- 厚労省「適正な優良事業者認定」

### 手数料（採用側）
- 公式公表なし。業界相場：年収の30～35%

## 5. 医療ワーカー（現・看護師ワーカー / 株式会社トライト）

### サービスの流れ
公式「サービスの流れ」ページの**5ステップ構成**（実質6～7段階）:

1. **登録**（フリーダイヤルまたはWeb、相談のみOK、全無料）
2. **ヒアリング**（登録時アンケート＋初回電話/対面の**2段階**）
3. **求人紹介**（地域最大級の情報量・電話/メール/LINE）
4. **面接対応**（スケジュール管理・手続き・書類アドバイス代行）
5. **条件調整**（給与・待遇・入退社日を交渉）
6. **就業・入職後支援**（転職後の悩み相談を継続受付）

### ヒアリング内容
2段階ヒアリング方式:
1. 登録フォーム（勤務地・勤務形態・希望年収）
2. 電話/対面での詳細ヒアリング（条件・現職の悩み・課題）

### 求人紹介
- **保有求人数**: 約65,000件
- **非公開求人**: 「看護師ワーカー専用の非公開求人」を多数保有
- **紹介手段**: 電話・メール・LINE
- 地域最大級の情報量を強調

### 書類サポート
- 履歴書・職務経歴書の準備サポート
- 添削
- 志望施設に特化した書類対策

### 面接対策
- 志望施設に特化した面接対策
- **面接同行あり**（対面・Web両方。面接会場に同席）
- 地方の転職でも対面ヒアリング・面接同行が可能

### 企業対応
- 面接日程調整代行
- 面接中に雇用条件・勤務内容を再確認
- 条件交渉代行（給与・役職・勤務条件）

### 内定後フロー
- 入退社日も交渉対象
- 採用条件確認書を作成・発行

### 入職後フォロー
- 「入職後に条件が違ったら相談を」
- **採用条件確認書と照らし合わせて改善サポート**（独自性）
- 定期フォロー実施

### 差別化ポイント
- **20年の実績**（看護師特化ネットワーク）
- **地方対応力**（対面ヒアリング・面接同行を地方でも実施）
- 管理職・講師など特殊求人を保有
- 大中小全規模の医療機関と取引
- 採用条件確認書による事後検証

### 手数料（採用側）
- 業界情報として「年収の30～35%が成功報酬」（axxisの解説記事）

## 共通業務フロー（5社の最大公約数）

5社すべてに共通して存在する11ステップ:

| # | ステップ | 共通する実施内容 |
|---|---------|----------------|
| 1 | **登録** | Web or 電話、1分以内で完了、無料、ゼロリスクで登録できる体験設計 |
| 2 | **初回面談（電話 or オンライン 30-60分）** | 登録後1～2営業日以内に担当CAが架電。電話・メール中心。マイナビのみ対面/Web面談にも対応 |
| 3 | **ヒアリング** | 希望条件（勤務地・勤務形態・給与・勤務時間）＋転職理由＋キャリアプラン＋不安。二段階式（看護roo!・医療ワーカー）と一段階式に分かれる |
| 4 | **求人紹介** | 公開＋非公開求人を複数提示。LINE/メール/電話。全社が非公開求人を保有すると訴求 |
| 5 | **応募書類作成** | 履歴書・職務経歴書の添削、志望動機作成支援 |
| 6 | **応募・面接調整** | 応募手続きと面接日程調整をCAが代行 |
| 7 | **面接対策** | 応募先ごとの想定質問、模擬面接（希望時）、マナー指導 |
| 8 | **面接同行 or リモート伴走** | レバウェル・看護roo!・ナースではたらこ・医療ワーカーは同行オプションあり。マイナビは明示なし |
| 9 | **条件交渉** | 給与・勤務条件・入職日の交渉をCAが代行（全社共通） |
| 10 | **退職交渉サポート** | 退職意向の伝え方・引継ぎ・スケジュール指導（全社共通） |
| 11 | **入職・定着フォロー** | 入職後の悩み相談。期間は各社でばらつき |

### 共通ヒアリング項目（最大公約数）
- 保有資格
- 勤務地・エリア希望
- 勤務形態（常勤・非常勤・夜勤有無）
- 希望年収
- 希望休日数
- 転職理由（現職への不満）
- 転職希望時期
- キャリアプラン（長期的な方向性）
- ライフプラン（子育て・結婚など）
- 現職の悩み・不安
- 連絡可能な時間帯

### 共通の「差別化訴求」トレンド
- 非公開求人の保有
- 独自取材による職場の内部情報
- LINE対応（ほぼ全社）
- 入職後フォロー

## AI代行化の視点での所見（調査から見えた示唆）

1. **ヒアリング項目は業界標準化されている** — 11項目程度で9割カバーできる。LINE BotでのQ&A完結が可能。
2. **「非公開求人」訴求は全社共通のマーケ文言** — 実質は「応募集中を避けるための運用」であり、データ構造的には公開求人と同じ。
3. **最強の差別化は「職場の内部情報」** — レバウェルの「年4,000回訪問」、看護roo!の「入職後アフターフォローからの情報逆流」が突出。ここはAIには難しい（=現場の足で稼ぐ領域）。
4. **面接同行は地方ほど価値が高い** — 医療ワーカーの地方対応が強いのはここ。神奈川集中なら同行コストは限定的。
5. **入職後フォローの「定期連絡スケジュール」は看護roo!が最も明確** — 当日/数日後/1週/1ヵ月/3ヵ月。AIで自動リマインド可能。
6. **退職代行・辞退代行は差別化されていない** — 全社がやっているが、マイナビと看護roo!のみ明示。ここはAI化しやすい（文面生成＋送信）。
7. **手数料30～35%は業界で硬直化** — 10%で殴り込むなら、CAの「人件費構造」を崩す必要。AIで大半を代行すれば原理的に可能。
