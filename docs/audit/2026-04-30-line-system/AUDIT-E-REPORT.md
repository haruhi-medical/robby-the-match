## E1. AI自動条件交渉文面 → 弁護士法72条 非弁行為リスク

### 該当箇所
- `aica-conversation-flow.md:783-875`（フェーズ18: 内定通知・条件交渉）
- AIが「交渉文面」「想定3パターン」「代替案」を自動生成し、社長承認後に病院送信

### 法的論点
**弁護士法72条**:
> 「弁護士又は弁護士法人でない者は、報酬を得る目的で訴訟事件、非訟事件…**その他一般の法律事件**に関して鑑定、代理、仲裁若しくは和解その他の法律事務を取り扱い…することを業とすることができない」

論点1: **賃金・賞与の交渉は「法律事件」「法律事務」にあたるか**
- 純粋な労働条件交渉は判例上グレー。報酬目的（紹介手数料発生）が絡む場合、業法該当性が議論される
- 「弁護士法72条違反の典型」としては破産手続代理・債務整理・離婚調停代理など
- 賃金交渉は「**労働関係法令の解釈・適用を伴うか**」で判断が分かれる

論点2: **人材紹介事業の付帯業務範囲**
- 職業安定法32条の3で定める手数料は「紹介の報酬」のみ。交渉代理は付帯業務として認められるかは整理必要

論点3: **AI生成文面の責任主体**
- AI出力 → 社長承認 → 病院送信、というフローでも、最終責任は事業者（はるひメディカルサービス）

### 推奨アクション

🔴 **社長アクション**:
1. **社労士＋弁護士に相談**: 「人材紹介事業の付帯業務として、賃金・賞与の交渉文面作成は適法か」
2. リスク許容できない場合、フェーズ18を **「客観データ提示までAI、文面起案は社長手動」** に変更（AIは「叩き台」を提示するが「送付文面」までは生成しない）
3. リスク許容できる場合、**重要事項説明** に「条件交渉支援はAI叩き台を社長承認のうえ実施。法的助言ではない旨」を明記

📝 **文書修正案**（aica-conversation-flow.md フェーズ18）:
```diff
-AI が客観的データ提示 + 交渉叩き台を生成（設計書 §5-8 参照）
+AI が客観的データ提示 + 交渉叩き台を生成。
+※ AI は法的助言を行いません。最終文面は社長判断で確定します。
+※ 賃金・労働条件の正式交渉は弁護士・社労士に依頼することをご案内します。
```

## E3. Q10「ご家族の状況と仕事との両立」→ 差別質問該当

### 該当箇所
- `aica-conversation-flow.md:719` フェーズ16 想定質問Q10

### 法的論点
厚労省「**公正な採用選考の基本**」ガイドライン:
> 採用選考時に配慮すべき事項として、**「家族に関すること（職業、続柄、健康、地位、学歴、収入、資産等）」「家庭環境などに関すること」は本来自由であるべき事項**

職安法5条の5（求職者等の個人情報の取扱い）:
- 業務目的の達成に必要な範囲を超えて個人情報を収集してはならない

男女雇用機会均等法5条:
- 募集・採用について、性別を理由とする差別的取扱いを禁止
- 「家族状況・両立可否」を女性により多く問うことは間接差別の疑い

### リスク
- 当社が「想定質問」として教えること自体は中立だが、**病院側が違法質問をすることを前提化**している印象
- もし病院担当者がこの質問をしたら、当社が「教育材料を提供した」責任を問われる可能性

### 推奨アクション

🟡 **即時文書修正**:

```diff
-Q10: ご家族の状況と仕事との両立について
+Q10: ライフスタイルと仕事との両立で大切にしたいこと
+※ 本来「家族構成・婚姻・育児」を直接問うことは公正な採用選考の観点から推奨されません。
+ もし聞かれた場合の応答練習として、「家族の有無に関係なく、長期的に働ける環境を重視」
+ 等、属性を伏せた回答を準備しておくと安心です。
```

または **削除**:
- Q10そのものを削除して9問に変更
- 代わりに「Q10: 入職後3年で達成したいこと」など中立的な内容に置換

   - 同調査では地域×産業のクロス公開は限定的

## E6. 退職交渉「引き止め対応集」の法的助言性

### 該当箇所
- `aica-conversation-flow.md:960-1021` フェーズ19 STEP 3「引き止め対応集」
- `aica-conversation-flow.md:979-980`「【パワハラ的引き止め: 紹介状を書かない等】→即座にお知らせください。労基・弁護士連携先をご案内します。」

### 法的論点
- 「引き止めへの返し方」自体は法的助言とまでは言えない（一般的なビジネスマナー）
- ただし「**紹介状を書かない**」等の具体的シナリオに対して「労基へ通報」「内容証明で退職届」等の助言をすると **法律事務該当の可能性**

### 推奨アクション

🟡 **文書修正で対処可能**:

📝 **修正案**:
```diff
-想定される引き止めと、返しをまとめました。
+想定される引き止めへの「一般的な伝え方の例」をまとめました。
+※ 以下は法的助言ではありません。具体的な対応に迷う場合は専門家にご相談ください。
```

```diff
-【パワハラ的引き止め: 紹介状を書かない等】
-→即座にお知らせください。労基・弁護士連携先をご案内します。
+【極端な引き止めに遭遇した場合】
```
