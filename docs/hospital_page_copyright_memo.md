# 病院別ページ生成の著作権・表記メモ

> 作成: 2026-04-18 / 次回実装前に要社長承認

## 実装想定

D1 `facilities` テーブル（29,549件）から主要病院を選定し、各病院について1ページ個別に生成する。

- 病院600〜1,000件（病床数100以上 / 看護師数50以上 等のしきい値で絞る想定）
- URL: `/lp/job-seeker/hospitals/<slug>.html`
- 各ページ: 病院概要 + 採用ポイント + 周辺情報 + FAQ + CTA

## 著作権・法的確認事項

### 1. 公的データの出典
- facilities テーブルの主要ソース: 厚労省 e-Gov 医療情報ネット、関東信越厚生局 指定一覧
- **公表情報＝事実データは再利用OK**（住所・電話・病床数・診療科・開設者）
- 各ページ下部に出典明記（例: 「厚生労働省 医療情報ネット 2026-03時点」）

### 2. 病院名・ロゴの扱い
- 病院名は事実表記のため掲載OK
- **ロゴ・画像・写真は必ず撮影者/権利者の許諾が必要** → 使わない方針が安全
- 代替: エリア地図や汎用SVGアイコンのみで構成

### 3. 求人情報の扱い
- ハローワーク公開求人は厚労省提供API（商用利用可）
- **件数・給与範囲の集計値**（「この病院の公開求人3件、月給25〜38万円」）は事実データなので掲載OK
- 個別求人の全文転載は**不可**（出典のハローワーク求人情報利用規約に従う）

### 4. 口コミ・評判の扱い
- SNSスクレイピングの病院口コミ掲載は**名誉毀損リスク**あり → 使わない
- 独自取材した看護師の声を掲載する場合は、本人の書面承諾が必須

### 5. 医療広告ガイドライン
- 医療機関の宣伝ではなく「看護師転職情報ページ」なので、医療法6条の2（医療広告規制）の直接対象ではない
- ただし「〇〇病院は優れた治療を提供」等の治療成果に関する表現は避ける
- 治療ではなく「看護師の働く環境（夜勤回数、有給取得率、教育体制など）」に焦点を当てる

## 推奨実装方針

### 段階的ロールアウト
1. **Phase 1**: 神奈川県の大規模病院 20-30件でパイロット
2. **Phase 2**: 社長が内容確認 + 法律家チェック
3. **Phase 3**: 東京/千葉/埼玉の主要病院へ展開

### ページ構成（テンプレート）
- Hero: 病院名 + エリア + 病床数 + 診療科サマリ
- 基本情報表: 病床数・看護体制・標榜診療科・最寄駅
- 看護師の働く環境: 公開されている場合のみ（研修、夜勤回数）
- 周辺環境: 駅、住環境、家賃相場
- この病院に強い転職相談: LINE CTA
- 出典: 厚労省データ URL + 最終確認日
- FAQ: 汎用3問
- **書かないこと**: 治療成果、ランキング、他院との優劣比較

### 必ず明示すること
- 最終データ確認日（データの鮮度）
- 厚労省出典URL
- 「本ページは公表情報に基づく情報提供であり、病院の広告ではありません」
- 許可番号 23-ユ-302928

## 社長承認が必要な判断ポイント

1. 病院名の公開表記 → OK（事実表記）
2. ロゴ・写真の使用 → **NO** の方針で良いか
3. 20-30件パイロットから始める段階的ロールアウトでよいか
4. 医療広告ガイドラインへのセルフチェック or 外部の法律家確認か
