# キャリアエージェント業務実態調査

> 作成日: 2026-04-21
> 調査目的: 大手人材紹介会社のキャリアエージェント（CA）の業務実態を解明し、AIで代行可能な領域を特定する
> 対象: 看護師紹介を中心に、人材紹介業界全体のCAの実態

---

## エグゼクティブサマリー（5行）

1. **CAは「電話工場」の労働者**。大手看護師エージェントでは1日80〜100件の架電ノルマがあり、未経験者のアポ獲得率は1〜2%。朝8時〜夜22時の拘束が常態化。
2. **ノルマは月150〜200万円の「粗利」**。年収500万円看護師1人成約で手数料100〜150万円（20〜30%）→ 月2〜3名成約が若手CAの最低ライン。
3. **レバウェル（旧レバレジーズメディカルケア）等の大手は固定残業80時間込みで月収28万円スタート**。平均残業51.4時間/月、営業職は21〜22時退社が普通、80時間超でも珍しくない。
4. **ツールはPORTERS／HRBC（2,200社導入）がデファクト**。架電にはMiiTel等のクラウドPBXで全通話録音・スコアリング、面談はZoom／TimeRex／Spir。求職者との連絡はLINE化が進行。
5. **「求職者より病院の都合」が構造的にビルトイン**。厚労省調査で医療機関の満足度は「不満」が7割、人材紹介経由の准看護師の1年以内離職率は41.5%、看護師は20.3%。2025年4月施行の法改正で祝い金と手数料不透明が一斉規制へ。

## A. 1日のルーティン

### 典型的タイムテーブル（大手人材紹介CAの標準例）

| 時刻 | 業務内容 | 具体的な量・内容 |
|------|---------|----------------|
| 8:00〜9:30 | 出社・メール／LINEチェック | 前日夜間に届いた求職者メッセージへの返信、朝会（15〜30分） |
| 9:30〜12:00 | 新規架電／求職者ヒアリング | 新規登録者への初回架電ラッシュ。1日80〜100件の架電目標のうち、午前だけで40〜50件かける日も |
| 12:00〜13:00 | 昼休み（実質スキップも多い） | 求職者対応で潰れがち |
| 13:00〜16:00 | 求人開拓／書類作成／面接同行／RAとの連携 | 求人票から3〜5件をピックアップして推薦、職務経歴書添削、企業との調整 |
| 16:00〜18:00 | 求人紹介・提案 | 求職者に電話で求人提示、応募意思確認 |
| 18:00〜21:30 | 就業後面談ラッシュ | 現職のある看護師との面談は「18時以降〜22時」に集中、オンライン面談が主流 |
| 21:30〜 | 面談記録入力・翌日の準備 | CRM入力、推薦文作成、翌日架電リストの作成 |

**出典根拠**:
- 1日100件架電＋朝8時〜22時勤務は大手エージェント経験者のnote証言（career-anchor.jp）。
- 「就業後（18〜20時以降）に面談が集中」は複数記事・openwork口コミで一致。
- 1日5〜6件の面談実施は大手CAの標準（circus-group.jp）。

### 繁忙期の違い

- **期末（3月・9月）、賞与支給後（7・12月）**: 看護師の転職活動が集中 → 架電ノルマが1.5倍に跳ね上がる期間あり。
- **連休明け月曜**: 週末に悩んで問い合わせてくる求職者への即レス対応が最優先。
- **夜勤明け対応**: 看護師特有で「夜勤明けの朝10時」に電話されて嫌がられる事例が多発（口コミ多数）。

## B. KPI・ノルマ

### 売上ノルマ

| レベル | 月間粗利（手数料収入）ノルマ | 月間決定人数目安 | 出典 |
|-------|----------------------------|----------------|------|
| 新人（1年目） | 100〜150万円 | 1〜2名 | talentsquare.co.jp |
| 若手（2〜3年目） | 150〜200万円 | 2〜3名 | talentsquare.co.jp |
| 中堅 | 300万円〜 | 3〜4名 | hrbc.porters.jp例示 |
| 大手エージェント標準 | — | 月8名決定期待 | ひとキャリ記事 |
| JACリクルートメント | 年間2,000万円（月166万円） | — | 業界基準値 |

**看護師業界の特殊事情**:
- 看護師の年収400〜500万円 × 手数料率20〜30% = 1名で80〜150万円の粗利。
- 月150〜200万円ノルマ = 最低月2名の成約が必要。

### 行動ノルマ（KPI）

**12種類のKPIが存在**（taqnoblog.com集計）:
1. 入金金額（月次）
2. 成約金額
3. 成約人数
4. 内定人数
5. 面接通過人数
6. 書類選考通過人数
7. 応募企業数
8. 推薦人数
9. 面談人数（月30名が中堅CAの水準）
10. 紹介獲得数
11. スカウト返信数
12. スカウト送信数

**具体的数値の目安**:

| KPI | 水準 | 出典 |
|-----|------|------|
| 架電数 | 80〜100件/日（大手看護師エージェント） | hop-job.com |
| 架電数 | 100件/日（一般人材紹介） | career-anchor.jp |
| アポ獲得率 | 2〜3.5%（テレアポ平均） | dgloss, comdesk |
| 新規面談数 | 月30名 | hrbc.porters.jp |
| 面談→決定率（著者実績） | 30〜35% | paopao-career.com |
| 内定承諾率（業界平均） | 60〜65% | porters |

### インセンティブ体系

- **基本給 + 歩合（粗利の数%〜10%）** が標準。
- レバレジーズ: 新卒月給30万円（固定残業80時間込み）→ 年収500万円スタート。成果によって1年目で800万円到達の事例あり。
- 平均年収513万円（レバウェル全社員平均、OpenWork）、天井は550万円前後との声。
- 賞与は「直近半年の成績」で決定、未達時は減額（ペナルティは少ないが「目標」として強く意識される）。

### 未達時の扱い

- 明確な降給・解雇は少ない（法的リスク）。
- ただし**賞与・昇給停止**、上司からの詰め、架電ノルマ増加、チーム内での肩身の狭さ。
- 3ヶ月連続未達 → 担当変更・内勤異動のケース。
- 結果的に**自主退職に追い込まれる構造**（開眼的な「辞めさせられる」ではなく「辞めたくなる」）。

## E. 必要スキル・知識

### 新人研修で仕込まれる内容

1. **ロープレ（ロールプレイング）** — 顧客役／求職者役を上司が演じて切り返しトーク練習。
2. **トークスクリプト暗記** — 初回架電、ヒアリング、クロージングの3段階スクリプト。
3. **AREA話法** — Assertion（主張）→ Reason（理由）→ Example（具体例）→ Re-assertion（再主張）。
4. **業界知識** — 医療業界の場合は「診療科、病床規模、教育体制（プリセプター制度）、二交代／三交代、看護基準」を最低限。
5. **法務** — 職業安定法、個人情報保護法、医療広告ガイドライン。

### クロージング話法・切り返しトーク

- 「他社と比較したい」 → 「他社にはない求人を優先紹介できます」
- 「まだ迷っている」 → 「◯◯さんの条件に合う求人は今月末で締切です」（緊急性演出）
- 「やっぱり転職やめます」 → 「一度面談だけでも」（粘る）

### 求人票から「やばい病院」を見抜くコツ（CAの暗黙知）

- **常時募集 × 大量採用** = 離職率が高い可能性。
- **「アットホーム」「風通しが良い」の乱用** = 人間関係が悪い可能性（言語化できない実態を隠すための定型句）。
- **「未経験歓迎」「ブランク歓迎」が過剰** = 定着しない。
- **給与幅が広すぎる**（例：月給22〜45万円）= 実際は下限提示。
- **病床機能報告データと比較して看護配置が薄い** = 1人当たり負担が重い。
- **GoogleMapクチコミ1.5以下** = 患者・家族からの評判も悪い。

### 医療業界知識の必須項目

- 急性期／回復期／療養／慢性期／精神／訪問看護の違い
- 7対1／10対1／13対1の看護基準
- プリセプター制度／クリニカルラダー
- 特定看護師・認定看護師・専門看護師
- 二交代（16時間夜勤）／三交代（準夜・深夜）の負担差

## F. 負の実態

### 「過酷」「ブラック」と言われる実態

- **残業**: レバウェル営業職の平均51.4時間/月、80時間込み給与、90時間超も珍しくない（OpenWork）。
- **勤務時間**: 営業は21〜22時退社が常態、求職者都合で日曜出勤も。
- **有給消化率**: レバウェル75.2%（相対的には良好）。
- **年収の天井**: 550万円で頭打ちとの証言複数。
- **月次リセット**: 毎月ノルマがゼロに戻る → 精神的圧迫が慢性化。

### 求職者から嫌われる理由

- **1日80〜100件の架電ノルマ** → 指定時間外の電話、夜勤明けの朝電話、執拗な連絡。
- **希望無視の求人紹介**: ノルマ達成のため「合格率が高い＝人手不足の病院」を優先紹介。
- **短期離職前提の押し込み**: 保証期間（3〜6ヶ月）を超えれば返金なしのため、入ってすぐ辞めても手数料は戻らない。
- **祝い金・引き抜き勧奨**: 2025年4月に全面禁止化されたが、裏で続行する業者が存在。

### 病院側（顧客）との板挟み

- 企業が求めるスペック「20代・経験5年以上・夜勤可・独身」vs 求職者希望「ワークライフバランス・日勤のみ」。
- 都内病院の人材紹介会社満足度「不満」7割（東京都病院協会調査、日経）。
- 手数料が高い（年収の20〜35%）→ 採用後すぐ辞められると病院は大損 → クレーム。

### ノルマのための「無理な応募押し付け」

- 月末に「とにかく推薦件数を増やす」→ 求職者の希望無視の大量推薦。
- 「1週間で内定出る病院」「即レスで決まる病院」を優先紹介 → ミスマッチ離職の温床。
- 業界調査: 人材紹介経由の准看護師1年以内離職率は41.5%（福祉医療機構）。

### 離職率・キャリアパス

- **人材業界全体の離職率は30%超の企業も少なくない**（paopao-career.com）。
- **3年以内離職率は50%超と言われる**（業界の通説、公式統計なし）。
- **典型的キャリアパス**:
  - 3〜5年CA経験 → 事業会社の人事、法人営業、独立起業、HR系ベンチャー転職。
  - 「CAスキルは汎用性が低い」との悩みが頻出。

## 結論：AI代行可能な領域の仮説（次フェーズ検討項目）

1. **1日100件の機械的架電 → AI音声Bot**（既存SaaSで実装可能）
2. **マッチング推薦（3〜5件）→ LLM + ベクトル検索**（Accel等は既に実装）
3. **推薦文作成 → ChatGPT APIで即自動化**
4. **日程調整 → TimeRex/Spirで人間不要**
5. **面談記録要約 → MiiTel + LLM**で既に一部実用化
6. **24時間LINE Bot対応 → 既存技術で実装済み（当社実装済み）**
7. **ノルマ管理・KPI可視化 → PORTERS等で既にSaaS化済み**

**AIで置換困難な領域**:
- 病院オーダーのイレギュラー対応（急募・特殊条件）
- 求職者の感情的動揺・家庭事情への傾聴
- 病院見学同行・面接同行
- 入職後のトラブル仲裁
- 「本当にやばい病院」の人脈経由情報収集

**差別化の核心**: 大手CAが「100件架電のノルマ消化」で疲弊する中、神奈川ナース転職は **「架電ゼロ × LINE Bot × 厳選3〜5件推薦 × 手数料10%」** で求職者ファースト体験を提供し、病院側も手数料負担軽減を両立できる。
