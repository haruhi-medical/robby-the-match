# Indeed掲載用 求人票テンプレート

## 掲載手順
1. https://jp.indeed.com/hire にアクセス
2. 無料アカウント作成（はるひメディカルサービスの事業者情報で）
3. 「求人を掲載」→ 以下の内容を入力
4. 応募方法を「外部サイトへのリンク」→ LINEリンクに設定

## 掲載のポイント
- 具体的な給与レンジを入れる（「応相談」はクリック率が下がる）
- 「ナースロビー」ではなく求人そのものの魅力を前面に
- 応募方法はLINEリンク（tel:は電話苦手な看護師が避ける）
- 無料掲載で十分。有料スポンサーは成約が出てから検討
