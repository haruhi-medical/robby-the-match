# 法的制約（絶対遵守）

## 職業安定法

- お祝い金・金銭提供による求職者勧奨 → **禁止**（2025年1月改正）
- 紹介により就職した者への2年間の転職勧奨 → **禁止**
- 虚偽の求人条件表示 → **禁止**（6ヶ月以下の懲役または30万円以下の罰金）
- 手数料実績の正確な表示 → **義務**

## 医療広告ガイドライン

- 特定医療機関への誘引性のある表現 → 規制対象の可能性
- 患者体験談の引用 → **禁止**

## SNS投稿

- すべてのストーリーは架空設定で作成する
- 患者の個人情報に触れない
- 実在施設の批判をしない
- ハッシュタグ5個以内（スパム判定回避）

## 個人情報保護

- 看護師の個人情報をSNSコンテンツに使用しない
- 病院の内部情報を特定可能な形でSNSに投稿しない
