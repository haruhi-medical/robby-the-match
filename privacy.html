個人情報についての約束

看護師さんの経歴や希望は、人生に関わる情報です。最小限・同意ベース・速やかに削除を原則とします。

  1. 必要な情報しか集めません。
    転職支援に不要な情報（家族構成、病歴、思想信条等）は取得しないよう努めます。
  2. 同意なく共有しません。
    求人医療機関への情報提供は、提供先・提供範囲をその都度確認した上で、同意をいただいてから行います。一斉送信は行いません。
  3. 辞めたら速やかに消します。
    退会のご連絡またはLINEブロックをもって削除意思とみなし、法令による保存義務がある記録を除き、原則30日以内に削除するよう努めます。
  4. いつでも確認・訂正・削除を請求できます。
    ご自身の情報が何に使われているかの開示請求に、手数料無料で対応します。

※ このセクションは当社の姿勢を示すものであり、本方針の法的条項は次項より始まります。

はるひメディカルサービス（以下「当社」）は、有料職業紹介事業者として、神奈川ナース転職／ナースロビー（以下「本サービス」）における個人情報の取り扱いについて、個人情報の保護に関する法律（以下「個人情報保護法」）その他関連法令を遵守し、以下のとおりプライバシーポリシーを定めます。

第2条（個人情報の定義）

本ポリシーにおける「個人情報」とは、個人情報保護法第2条に定める個人情報を指し、氏名、生年月日、住所、電話番号、メールアドレス、その他の記述等により特定の個人を識別できる情報（他の情報と容易に照合することができ、それにより特定の個人を識別することができることとなるものを含む）を指します。

第3条（個人情報の収集方法）

当社は、以下の場合に適法かつ公正な手段により個人情報を取得します。

  • 本サービスの登録フォームへの入力時
  • AIチャットボットとの会話時
  • 電話・メールでのお問い合わせ時
  • 面接・見学の日程調整時
  • 求人医療機関からの採用結果等の報告時

なお、看護師免許等の資格情報については、職業紹介サービスの提供に必要な範囲で、ご本人の同意を得た上で取得します。

第4条（個人情報の利用目的）

当社は、収集した個人情報を以下の目的で利用します。利用目的の範囲を超えて個人情報を取り扱うことはありません。

  1. 求職者への求人情報の提供および有料職業紹介サービスの提供
  2. 求人医療機関への求職者情報の提供（本人の同意を得た場合に限る）
  3. 面接日程の調整、入職手続きの支援
  4. 入職後のフォローアップ（定着支援）
  5. サービスに関するご連絡・お問い合わせへの対応
  6. サービスの改善・新サービスの開発
  7. 統計データの作成（個人を特定できない形式に加工した上で）
  8. 職業安定法その他関連法令に基づく記録の作成・保存

第5条（個人情報の第三者提供）

当社は、以下の場合を除き、本人の同意なく個人情報を第三者に提供しません。

  • 求職者本人の同意がある場合（求人医療機関への情報提供）
  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
  • 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

求人医療機関への情報提供にあたっては、以下の原則を遵守します。

  • 提供する情報の範囲（氏名、経験年数、資格、希望条件等）を事前にご本人に確認し、書面またはデジタル上で同意を取得します。
  • 同一案件について、同意のない複数の求人医療機関への一斉送付は行いません。
  • ご本人からの提供停止・取り下げの依頼には、合理的な期間内に対応するよう努めます。

第6条（個人情報の安全管理措置）

当社は、個人情報の漏洩、滅失、毀損等を防止するため、以下の安全管理措置を講じます。

  • 組織的措置：個人情報保護管理者の設置、取扱規程の整備、従業員への定期的な教育・研修
  • 技術的措置：アクセス制御、通信の暗号化（SSL/TLS）、情報システムへの不正アクセス防止措置
  • 物理的措置：個人情報を取り扱う機器・書類の施錠管理
  • 人的措置：個人情報を取り扱う従業員に対する守秘義務の徹底および適切な監督

第7条（個人情報の保存期間）

当社は、個人情報を利用目的に必要な範囲内で保存します。具体的な保存期間は以下のとおりです。

  • 求職者登録情報：紹介サービス完了後2年間（職業安定法に基づく記録保存義務期間を含む）
  • AIチャットボットの会話記録：サービス提供完了後1年間
  • お問い合わせ記録：対応完了後1年間

保存期間経過後は、速やかに個人情報を削除または匿名化処理いたします。なお、法令により保存が義務付けられている情報については、当該法令の定める期間保存します。

退会（本サービスの利用停止）のご連絡をいただいた場合、およびLINE公式アカウントのブロックにより明示的な削除意思表示があったとみなせる場合、法令上の保存義務対象を除き、原則として30日以内に個人情報を削除するよう努めます。

第8条（個人情報の開示・訂正・削除等の請求）

本人から保有個人データの開示、訂正、追加、削除、利用停止、第三者提供の停止等（以下「開示等」）の請求があった場合、本人確認の上、合理的な期間内に対応いたします。

開示等の請求手続は以下のとおりです。

  1. 下記お問い合わせ窓口にメールまたは電話にてご連絡ください。
  2. 当社所定の請求書をお送りしますので、必要事項をご記入の上、本人確認書類とともにご提出ください。
  3. 本人確認の上、原則として2週間以内に対応いたします。

開示等の請求に際して手数料はかかりません。

第9条（Cookie・外部送信ツールの使用）

本サービスでは、サービス向上および広告効果の測定のため、以下の外部ツールを使用しています。これらのツールはCookieや類似技術を通じて利用者の閲覧情報を収集し、各ツール提供事業者に送信します（改正電気通信事業法第27条の12に基づく外部送信規律への対応）。

（1）Google アナリティクス 4（GA4）

  • 提供事業者：Google LLC
  • 収集情報：ページ閲覧履歴、滞在時間、流入元、デバイス情報、おおよその位置情報
  • 利用目的：ウェブサイトの利用状況の分析・改善
  • オプトアウト：Google アナリティクス オプトアウト アドオンをインストールしてください
  • プライバシーポリシー：Google プライバシーポリシー

（2）Meta ピクセル

  • 提供事業者：Meta Platforms, Inc.
  • 収集情報：ページ閲覧履歴、ボタンクリック等のイベント情報、デバイス情報
  • 利用目的：Instagram広告の効果測定およびリターゲティング広告の配信
  • オプトアウト：Facebookの広告設定から変更できます
  • プライバシーポリシー：Meta プライバシーポリシー

（3）Microsoft Clarity

  • 提供事業者：Microsoft Corporation
  • 収集情報：マウス操作、スクロール、クリック等の操作情報、ページ閲覧履歴、デバイス情報
  • 利用目的：ウェブサイトのユーザビリティ改善（ヒートマップ・セッション録画による分析）
  • オプトアウト：ブラウザのCookie設定から第三者Cookieを無効にすることで停止できます
  • プライバシーポリシー：Microsoft プライバシーステートメント

上記ツールにより収集される情報は、単独で特定の個人を識別するものではありませんが、Cookieを通じてブラウザを識別し、行動履歴を分析するために使用されます。ブラウザの設定によりCookieの受け入れを拒否することも可能ですが、一部のサービス機能が制限される場合があります。

第10条（AIチャットボットにおける個人情報の取り扱い）

本サービスのAIチャットボットの利用に関し、以下のとおり個人情報を取り扱います。

  1. AIチャットボットとの会話内容は、職業紹介サービスの提供および品質向上の目的で記録・保存されます。
  2. 会話開始時に利用目的を明示し、同意を取得した上で記録を行います。同意しない場合、AIチャットボットの利用を中止できます。
  3. AIチャットボットが提供する情報は参考情報であり、正式な雇用条件を保証するものではありません。
  4. 会話記録の保存期間はサービス提供完了後1年間とし、期間経過後は削除いたします。
  5. 会話記録の削除を希望される場合は、下記お問い合わせ窓口までご連絡ください。
  6. AIチャットボットの会話データは、外部のAIサービス提供事業者（OpenAI社のGPT）のAPIを通じて処理されます。また、一部のコンテンツ生成処理においてCloudflare Workers AIを利用する場合があります。これらのAIサービスに送信されるデータには、氏名・電話番号等の直接的な個人識別情報は含まれません。AIはテキストの構成・改善のみに使用されます。送信データはAIモデルの学習には使用されず、APIログは最大30日間保持後に削除されます。

第11条（AI履歴書作成機能における個人情報の取り扱い）

本サービスの「AI履歴書作成機能」の利用に関し、以下のとおり個人情報を取り扱います。本機能のご利用には、フォーム送信前の明示的な同意が必要です。

  1. 本機能では、履歴書に記載する氏名、ふりがな、生年月日、性別、電話番号、メールアドレス、住所、学歴、職歴、資格、志望動機のヒント、本人希望を入力いただきます。
  2. 入力情報のうち、志望動機欄の文章生成のために、氏名・年齢・職歴の概要・資格・志望動機のヒントを OpenAI 社（米国）の GPT API に送信します。これは個人データの外国にある第三者への提供（米国への越境移転）に該当するため、事前に明示的な同意を取得します。
  3. OpenAI 社に送信されたデータはAIモデルの学習には使用されず、APIログは最大30日間保持後に削除されます（OpenAI API のデータ保持ポリシーに準拠）。
  4. 生成された履歴書HTMLは、当社が利用する Cloudflare 社のストレージ（KV）に暗号化通信で保存され、作成から7日間で自動削除されます。
  5. 履歴書の閲覧URLは、ランダムに生成される36桁の推測不能な識別子を含み、作成者本人とご担当者以外には共有されません。
  6. 入力情報および生成された履歴書の削除を希望される場合は、下記お問い合わせ窓口までご連絡ください。
  7. 本機能の利用にあたり同意されない場合、AI履歴書作成機能の利用を中止することができます（他の求人紹介機能は同意の有無に関わらずご利用いただけます）。
