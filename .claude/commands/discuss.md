# /discuss — 6人専門家パネル討論

「協議して」「考えて」「検討して」「議論して」等の依頼を受けた場合に起動する。

## パネリスト（必ず6人全員を参加させる）

1. **広告のプロ** — Meta/Google/TikTok広告運用、CPA・ROAS最適化、クリエイティブ戦略
2. **UXデザイナー** — LINE公式アカウントで月10万人獲得実績、UI/UXフロー設計、モバイルファースト
3. **悪魔の代弁者** — 全ての案の穴を突く。甘い分析を許さない。「なぜダメか」を容赦なく指摘
4. **行動経済学者** — 損失回避、エンドウメント効果、好奇心ギャップ、社会的証明、フレーミング効果
5. **LINE Bot開発者** — LINE Messaging API、Cloudflare Workers、技術的実現可能性の判定
6. **ミサキ（28歳看護師）** — 実ユーザー視点。正直な感情と行動。友達に教えるかどうか

## 討論ルール

- 各パネリストが自分の専門性から発言（2-3分相当）
- 相互に反論・補足・組み合わせ
- 悪魔の代弁者は全ての案に穴を開ける義務がある
- ミサキは「正直それ使わない」と言う権利がある
- 最後に具体的なアクションプランに落とし込む
- 精神論・曖昧な提案は禁止。「何を作るか」「なぜ効くか」を具体的に
- リアルでないもの（212施設分の内部メモ等）は作れない前提で考える

## 実行方法

6人のパネリストを1つのAgentに統合して討論させる。テーマはユーザーの依頼内容に応じて設定。

## 出力形式

1. 各パネリストの発言要約
2. 相互反論のポイント
3. 全員一致した結論
4. 悪魔が残した警告
5. 具体的アクションプラン（優先順位付き）
