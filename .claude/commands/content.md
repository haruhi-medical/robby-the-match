# /content — TikTok/Instagram コンテンツ生成

SNSコンテンツを生成してキューに追加する。

## 引数
- 指定なし: TikTok用スライドショー台本1本生成
- `tiktok N`: TikTok台本をN本生成
- `instagram`: Instagram用カルーセル1本生成
- `batch`: TikTok3本 + Instagram1本のバッチ生成

## 手順

1. `posting_queue.json` の現在のキュー状態を確認
2. `ai_content_engine.py` のコンテンツMIX比率を確認:
   - あるある35% / 給与20% / 業界裏側15% / 地域15% / 転職10% / トレンド5%
3. 不足しているカテゴリを優先して台本生成
4. フック品質チェック（25文字以内、5W1H必須、オープンループ、事実ベース）
5. `generate_carousel.py` でスライド画像生成
6. `posting_queue.json` にキュー追加
7. 生成結果のサマリを表示

## フックの品質基準
- **25文字以内**: 省略しすぎず日本語として自然
- **5W1H**: 誰が・何が・未解決の問い・解釈が1つ
- **オープンループ**: 疑問（？）/未完了（...）/理由/なぜ で終わる
- **事実ベース**: 数字は実際のデータ。嘘禁止
- **多様性**: バッチ内で同じ数字/テーマの繰り返し禁止

## CTA 8:2ルール
- 10投稿中8本: ソフトCTA（保存、フォロー）
- 10投稿中2本: ハードCTA（LINE誘導）
