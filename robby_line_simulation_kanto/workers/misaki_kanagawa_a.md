# Misaki Kanagawa-A: LINE Bot Simulation (MK-K-A-001 ~ MK-K-A-050)

> **Scope:** 神奈川県A（横浜・川崎 20件 / 湘南・鎌倉 15件 / 相模原・県央 15件）
> **Persona:** ミサキ（28歳看護師）— 二交代制、急性期病棟4年目
> **DB:** 神奈川5,165施設
> **Sub-areas:**
> - 横浜・川崎: 横浜市, 川崎市 — `il_area=yokohama_kawasaki` → `['横浜', '川崎']`
> - 湘南・鎌倉: 藤沢市, 茅ヶ崎市, 鎌倉市, 逗子市, 葉山町, 寒川町 — `il_area=shonan_kamakura` → `['藤沢', '茅ヶ崎', '鎌倉', '寒川', '逗子', '葉山']`
> - 相模原・県央: 相模原市, 厚木市, 海老名市, 座間市, 綾瀬市, 大和市, 愛川町 — `il_area=sagamihara_kenoh` → `['相模原', '厚木', '海老名', '座間', '綾瀬', '大和', '愛川']`
> **Phase Flow:** il_area → il_subarea → il_facility_type → (il_department) → il_workstyle → il_urgency → matching_preview → matching_browse → handoff

6. **handoff後のBot沈黙（Medium）** — MK-K-A-031: handoff_silent中のユーザーメッセージに対する自動応答がない。
7. **居住地と勤務地の不一致対応（Medium）** — MK-K-A-019, 050: 居住地情報を収集しないため、通勤距離でのソートが不可能。
